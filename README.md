# Kali Termux Custom Setup

## Installation

## package update
```
pkg update -y && pkg upgrade -y

```
## install git
```
pkg install git -y
```
## git clone

```
git clone https://github.com/MHAleedduuub/Termux

```
# cd file for execute
```
cd setup.sh
```
## chmod

```
chmod +x setup.sh
```
## setup
```
./setup.sh
```

# now you have kali linux on your termux
